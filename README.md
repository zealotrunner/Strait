Strait
======

Trait simulation for PHP 5.3


Usage
-----


Test
----

Install [Composer](https://github.com/composer/composer)
```shell
cd yapo
curl -sS https://getcomposer.org/installer | php
php composer.phar install --dev
```

Prepare for testing
```shell
./vendor/bin/phpunit tests/StraitTest.php
```
